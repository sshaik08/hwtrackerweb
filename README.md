# Homework Tracker Project #

### A web-based homework tracker that allows the user to track their assignments ###

This was a collaborative project in which my peer and I developed a site for students to be able to keep track of their assignments. Our website allows students to
- record the due date, subject, assignment name, and completion status
- add additional rows for assignments as needed
- save the current tracker to their device to be opened later
- open a previous version of the tracker and make updates as needed
